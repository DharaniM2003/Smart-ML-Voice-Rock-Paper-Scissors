# Smart-ML-Voice-Rock-Paper-Scissors



## 📌 Project Overview
Smart ML Voice Rock-Paper-Scissors is an AI-powered game that combines voice recognition with Machine Learning to create an interactive gaming experience. The application listens to the player's voice commands, predicts game outcomes, tracks performance, and adapts gameplay through intelligent decision-making.

---

## 🎯 Problem Statement
Traditional Rock-Paper-Scissors games rely on manual input and fixed logic. This project introduces voice-controlled gameplay with Machine Learning, making the game more interactive, engaging, and intelligent.

---

## 📂 Features
- 🎤 Voice Command Recognition
- 🤖 Machine Learning-based Decision Making
- 🎮 Rock, Paper & Scissors Gameplay
- 📊 Score & Statistics Tracking
- 🏆 Level Progression System
- 🎵 Background Music Support
- 💻 Interactive User Interface

---

## 🛠 Technologies Used

---

## 🚀 Project Workflow
1. Capture Voice Command
2. Convert Speech to Text
3. Identify Player's Move
4. ML Model Predicts Computer Move
5. Display Winner
6. Update Score & Statistics
7. Continue Level Progression
